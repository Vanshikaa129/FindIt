# FindIt – Lost & Found Web App

**FindIt** is a simple and user-friendly **Lost & Found frontend web application** that helps users report lost items and find items that have been found.  
This project is built to demonstrate **HTML, CSS, and JavaScript skills** in a real-world use case.

---

##  Features

-  Report lost items with details
-  View list of found items
-  Search items by name or category
-  Clean and responsive UI
-  Fast and lightweight (no backend)

---

##  Tech Stack

- **HTML** – Structure  
- **CSS** – Styling & layout  
- **JavaScript** – Logic & interactivity
